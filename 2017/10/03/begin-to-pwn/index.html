##0x01.环境搭建
虚拟机:virtualbox 或者 vmware
os：ubuntu16.04（也可以直接用kali）
环境配置

  • shell——–>zsh（oh-my-zsh）
    sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)" 提高生产效率

  • editor——>sublime(atom or vim)
    官方安装包+百度密钥

-script env—>python(ipython)
apt-get install python-dev libssl-dev libffi-dev
exp编写主流脚本语言 ipython是一个更易于使用的python交互式解释器

  • python lib—>pwntools&&zio
    两大主流pwn库 建议仔细阅读官方文档
    pip install zio pip install ipython

  • 32lib———-> lib32readline-gplv2-dev
    添加32位的支持 保持使用一个主系统
    `apt-get install build-essential module-assistant apt-get install gcc-multilib g++-multilib apt-get install lib32readline-gplv2-dev

  • socat—->将程序在本地端口运行socat tcp-l:6666,reuseaddr,fork exec:./pwn
    apt-get install socat

  • gdb plugin–>peda pwndbg
    必备gdb插件 增加更多可视化信息 内置很多强大外挂
    https://github.com/longld/peda https://github.com/pwndbg/pwndbg

  • ida–>居家旅行必备，最近leak出了7.0，f5大法好。
    • 插件 keypatch
  • 吾爱破解工具包
  • 系统自带工具
    • readelf（elf文件格式分析）
    • objdump（反汇编elf文件）
    • ldd（查看程序依赖库）
    • ltrace（跟踪库函数调用）
    • strace（跟踪程序调用）
    • file(查看文件类型)

**##建议手动搭建一遍环境 熟悉作业环境 阅读各种工具官方文档或者是帮助文档 能够善用佳软

##0x02 hello world

  • protostar——>入坑练习镜像
    • 有5部分组成
      • stack 练习（栈溢出）
      • fsb 练习（格式化字符串漏洞）
      • heap 练习（堆溢出）
      • net 练习 （网络编程）
      • final 练习 （模拟真实题目）
  • ctf 真题
    • stack（貌似基本没有单纯的栈溢出）会伴随nx（rop） canary（leak||brute） alsr（leak）
    • rop–>ret2libc
      -srop–>利用linux的信号机制
    • ret2dlreslove–>利用lazy binding机制
    • brop –>盲打暴力寻找gadget
      -FSB（说好的比较偏 但是为什么经常要用到）
      • 传统型 直接改写got表
      • 非传统 改写全局变量
      • 辅助型 用于leak
    • heap（相当于数学大题）fengshui（堆排布）—->改写程序流
